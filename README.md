# hubot-poker

Allow hubot to play a game of holdem.

See [`src/poker.coffee`](src/poker.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-poker --save`

Then add **hubot-poker** to your `external-scripts.json`:

```json
["hubot-poker"]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
